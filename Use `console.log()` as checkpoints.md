This is one of the simplest and most powerful tools.
Example:

```js
function compareValues(a, b) {  
	console.log("a =", a);  
	console.log("b =", b);  
  
	if (a > b) {  
		console.log("Entered first branch");  
		window.alert(a + " is greater than " + b);  
	} else {  
		console.log("Entered else branch");  
		window.alert(a + " is less than " + b);  
	}  
}
```
Now you can see:
- what values the function received
- which branch ran
- whether your assumptions were wrong
    
I recommend this constantly while learning. It is not “beginner stuff.” Professionals do this too.
