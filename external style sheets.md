user-defined style sheets can be superseded by external styles, which are styles created and placed within a CSS file and linked to the website. 

You used external style sheets in the last tutorial when you linked the Curbside Thai website to a collection of CSS files. As you saw in that tutorial, multiple documents can access the same style sheet, making it easier to apply a common design to an entire website.
