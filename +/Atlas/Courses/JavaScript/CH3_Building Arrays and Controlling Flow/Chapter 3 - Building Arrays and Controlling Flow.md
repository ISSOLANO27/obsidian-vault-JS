## Introduction

The code you have written so far has been linear in nature. In other words, your programs start at the beginning and end when the last statement in the program executes. However, **sometimes it can be useful to change this default order of execution.** Changing the order in which JavaScript code is executed is known as controlling flow. Controlling the flow of code is one of the most fundamental skills required in programming. Before learning how to control program flow, you will first learn about the array data type, which is often used with that task.
