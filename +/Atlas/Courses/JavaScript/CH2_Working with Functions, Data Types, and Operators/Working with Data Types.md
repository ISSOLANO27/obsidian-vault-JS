Variables can contain many different kinds of values such as the time of day, a dollar amount, or a person’s name. A [[data type]] is the specific category of information that a variable contains. The data type determines how much memory is allocated for the data stored in the variable. The data type also governs the kinds of operations that can be performed on the variable.

Many programming languages require that you declare the type of data that a variable contains. 
Such languages are called [[Strongly typed programming languages]]. #stronglyTyped

 Programming languages that do not require you to declare the data types of variables are called [[Loosely typed or Duck typed programming language]]s.
