In your code, you will often need to run functions and respond to events occurring within the web page or browser.

There are **3 ways of associating a function with an event**: 
-  adding an attribute to an HTML tag, 
-   adding a property to a page object, or 
-  attaching an event listener to a page object

Each has its **advantages** and **disadvantages**
