Most web pages include [[Embedded content]], 

>Inline images and contact info live comfortably in the **content layer**.  
Hypertext and event handling pull them into the **behavior layer**.

## Embedded Elements
These are also known as [[Interactive Elements]]

| Element                     | Description                                                                    |
| --------------------------- | ------------------------------------------------------------------------------ |
| [[audio]]                   | Represents a sound clip or audio stream                                        |
| [[canvas]]                  | Contains programming scripts used to construct bitmap images and graphics      |
| [[embed]]                   | Contains general embedded content including application or interactive content |
| [[iframe]]                  | Contains the contents of an external web page or Internet resource             |
| [[img]] **(inline images**) | Contains a graphic image retrieved from an image file.                         |
| [[object]]                  | Contains general embedded content including application or interactive content |
| [[video]]                   | Represents a video clip or video stream with captions                          |
 These are also known as `interactive elements` since they allow for interaction between the user and the embedded object. Think player buttons to control the playback. 

>Always include the `alt` attribute;` it is REQUIRED in XHTML`. Most important it accommodates users that run nonvisual web browsers.
>
#OlderVersionOfHTML 
---

```html
<img src="file" alt="text"/>
```

`file` is the name of the image file, if the browser cannot display the `alt` text is used instead of the image.
