### Case: _Pandaisia Chocolates_

Anne Ambrose is the owner and head chocolatier of _Pandaisia Chocolates_, a chocolate shop located in Essex, Vermont. You have been asked to assist on the redesign of the company’s website. Anne has provided you with three pages from the website to start your work. She has written all of the content, compiled the necessary images and graphics, and written some of the text and color styles. She needs you to complete the project by designing the page layout using the CSS layout properties.
