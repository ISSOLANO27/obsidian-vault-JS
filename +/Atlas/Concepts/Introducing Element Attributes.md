Attributes provide additional information about the purpose of the element.

```html
<element attr1="value1">content</element>
```

Rules:
1. can be listed in any order
2. must come after the element name
3. if more than one, must be separated by a space.
4. each value must be enclosed in " "

> For those that `do not` require one, HTML supports `attribute minimization` by removing the attribute value completely.

```html
<p hidden>content</P>
```
