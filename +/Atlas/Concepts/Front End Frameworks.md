These are Client side [[Web Framework]]s. Brain of the web page.

When someone visits your website:

1. The server sends HTML, CSS, and JS
2. The browser downloads it
3. AngularJS executes inside the browser

What does it do there?

It controls:

- UI
- Dynamic updates without reloading
- Data binding (when you change data, the screen updates automatically)
- 
It **DOES NOT** talk to the Database directly
