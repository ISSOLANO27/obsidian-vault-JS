## Fixed are defined in pixels (px)
the widths of the columns and margins are specified in `pixels`, where every column has a fixed position. 

## Fixed stay at 960px
Many fixed grid layouts are based on a page width of 960 pixels because most desktop screen widths are at 1024 pixels (or higher) and a 960-pixel width **leaves room for browser scrollbars** and other features. 

### Why 960 Pixels Work Well for Layouts
The 960-pixel width is also easily divisible into halves, thirds, quarters, and so forth, making it easier to create evenly spaced columns. [[960 divisibility table]]

> Problem: it does not account for other screen sizes
