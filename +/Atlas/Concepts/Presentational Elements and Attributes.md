Earlier versions of HTML supported these.

They use to describe how each element should be rendered by the browser. 

if you wanted to align text on a page:
```html
<element align="alignment">content</element>
```

> Alignment would be wither `left`, `right`, `center` or `justify`.
> Almost all presentational elements and attributes are now *deprecated* in favor of style sheets
