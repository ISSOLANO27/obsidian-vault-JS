To jump to a specific location within a document, identify that location by adding the id attribute to an element tag at that location.
```html
id="text"
```
```html
<h2 id="lunch">Lunch Menu</h2>
```

>NOTE: Id MUST be unique. If you assign the same one to another element, then Browser will render the first one it encounters. #FIFO

- IDs are case sensitive: top != TOP
