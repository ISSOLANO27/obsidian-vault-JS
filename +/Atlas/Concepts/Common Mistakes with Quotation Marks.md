A common mistake is to mix the two types of quotation marks or omit one of them as in the following statement in which a double quote begins the text string, but a single quote ends it:
```js
document.write('Welcome to Fan trick photography');
```
> You must always match a beginning quote with an ending quote of the same type or else an error will result in your code. Without matching quotes, the JavaScript interpreter cannot tell where the text string ends. #writingEffectiveCode
