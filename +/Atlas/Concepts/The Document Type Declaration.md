<!DOCTYPE html> is used as a processing instruction indicating the markup language used in the document.

Browser uses this to know which standard to use for displaying content.

Older versions of HTML are very complicated:
`<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN""http://dwww.w3.org/TR/html4/strict.dtd">  

#OlderVersionOfHTML

Others did not even need them. Often indicating page should be rendered in `quirks mode` (1900s-2000s)

When present it would render in `standard mode`. (*most current specifications*)

Always use, <!DOCTYPE html>
