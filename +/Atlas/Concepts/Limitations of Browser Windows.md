While you can do a lot with browser windows, they do suffer from some very important limitations:
##### can be blocked by popup blockers
- Popup browser windows can be blocked by popup blockers installed by the user or built into the browser itself.
##### do not scale well to small mobile devices
- Browser windows do not scale well to small mobile devices such as cell phones, which put a premium on screen space.
##### cannot display files that are loaded locally
- For security reasons, browser windows cannot display files that are loaded locally on the user’s computer.
##### support might not be available
- Browsers are not consistent in how they handle browsers windows. A feature that is supported in one browser might not be supported in a different browser.
    

All of this is not to say you should never use a browser window as a popup. *They are flexible and easy to apply* and make sense for some applications. However, their use has **declined** in recent years due to their limitations.

#### Note
##### user initiated events are the exception
Popup blockers generally will not block popup windows that are opened in response to user-initiated events, such as clicking a form button.
