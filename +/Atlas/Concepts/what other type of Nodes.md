
You already know:

- Elements (`<div>`)
    
- Text (`"Hello"`)
    

Here are the rest (this is exam / pentester knowledge):

---

### 🔹 Common node types:

|Type|Example|
|---|---|
|Element Node|`<p>`|
|Text Node|`"Hello"`|
|Comment Node|`<!-- comment -->`|
|Document Node|whole page|
|DocumentFragment|temporary container|
|Attribute Node|`class="box"`|

---

### 🧠 Quick visual:

```html
<div>
  Hello
  <!-- comment -->
</div>
```

Breaks into:

- `<div>` → element node
    
- `"Hello"` → text node
    
- `<!-- comment -->` → comment node
