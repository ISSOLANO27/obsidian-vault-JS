used for items that follow some defined sequential order. Alphabetically or numerically.

```html
<ol>
	<li>item1</li>
	<li>item2</li>
</ol>
```
You could also add the [[value]] attribute to each list item:

```html
<ol>
	<li value="49">California</li>
</ol>
```
#groupingElements 
By default each item will appear on a new line in the document.

Attributes:
- [[start]]
- [[reversed]]
