Exponential values in JavaScript are written with the letter `e` separating the decimal and exponent parts of the value, such as 2.0e6 for the value  or 2,000,000, or  for  or 0.00031. 

## **Beyond the limits**:

Using exponential notation, JavaScript can represent floating point values as large or small as approximately  or as small as. 

- Numbers larger than 1.8×10308 are automatically converted to the special value `Infinity`.
    
- Numbers smaller than −1.8×10308 become `-Infinity`.
    
- These are not numeric values in the usual sense but special keywords that indicate overflow.
