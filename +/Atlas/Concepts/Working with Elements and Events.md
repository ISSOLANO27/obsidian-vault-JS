## Click Event again..

The [[click event]] is available for a number of elements, including `a` elements and `form controls` that are created with the `input` element.

This will NOT work on a body element. It does not  have a click event. 
It does have a [[load]] event and an [[unload]]

---

There are a few [[Different Ways to Specify Event handlers]] an [[event handler]] for a particular event. One way is to include event handler code as an attribute of the element that initiates the event
