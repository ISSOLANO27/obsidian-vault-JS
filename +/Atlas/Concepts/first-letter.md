```css
blochquote p::first-letter {
	font-family: 'Times New Roman', Times, serif;
	font-size: 250%;
}
```

This matches the **first letter of every paragraph within a blockquote** and displays the character in `Times New Roman` font to `250%` larger than the surrounding text.


> The `::` separator was introduced to CSS to **differentiate** pseudo-elements from pseudo-classes.
