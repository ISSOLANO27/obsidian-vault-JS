Exception handling provides a graceful way to handle errors, **especially user error**. 

In addition to handling errors within a specific section of code, JavaScript allows the programmer to create custom methods for handling any errors that may appear anywhere within the program. 

Many programmers prefer to write their own error handlers in place of the default error handlers built into the browser and viewed within the debugger console.
