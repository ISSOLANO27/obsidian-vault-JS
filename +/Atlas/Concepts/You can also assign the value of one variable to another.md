In the following code, the first statement declares the `salesTotal` variable without assigning it an initial value, the second statement declares the `curOrder` variable with an initial value of `47.58`, and the third statement assigns the value of the `curOrder` variable to the `salesTotal` variable.

```js
let salesTotal; // declare empty variable
let cutOrder = 47.58;
salesTotal = cutOrder // assign value to empty variable
```
