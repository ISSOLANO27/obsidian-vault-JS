Once you’ve marked the location with an ID, you link to that element using the following hypertext link: #aAnchors 
```html
<a href="ct_fullmenus.html#lunch">View our Lunch Menu</a>
```
#linkWithId
 To link to a location within the current page, include only the ID value along with the `#` symbol.
 ```html
 <a href="#lunch">View our Lunch Menu"</a>
 ```
> In both cases, clicking or tapping the link will cause the browser to automatically scroll to the location within the page.
