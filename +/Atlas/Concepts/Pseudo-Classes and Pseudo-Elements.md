Not everything that appears in the rendered page is marked up in the HTML file. 

For example, a paragraph has a first letter or a first line but those are not marked as distinct elements. 
Similarly, an **element can be classified based on a particular property without having a class attribute**. 

The initial entry from an ordered list has the property of being the first item, but no class attribute in the HTML file identifies it as such. These **elements and class attributes that exist only within the rendered page but not within the HTML document are known** as `pseudo-elements` and `pseudo-classes`. 

>Despite not being part of the HTML document, you can still write style rules for them.
