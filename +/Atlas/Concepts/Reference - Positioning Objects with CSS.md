

## To shift an object from its default position, use the properties
    
```css
position: relative;
top: value;
bottom: value;
left: value;
right: value;
```
    
where _value_ is the distance in one of the CSS units of measure that the object should be shifted from the corresponding edge of its container.
    
## To place an object at a specified coordinate within its container, use the properties
    
```css
position: absolute;
top: value;
left: value;
bottom: value;
right: value;
```
    
where _value_ is a distance in one of th-e CSS units of measure or a **percentage** of the container’s width or height.
    
## To fix an object within the browser window so that it does not scroll with the rest of the document content, use the property:
```css
position: fixed;
```
