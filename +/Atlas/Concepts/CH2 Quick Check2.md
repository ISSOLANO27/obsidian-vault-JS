1. What is the difference between block scope and function scope?
    
    [Variables declared with let are block scoped in that their scope is limited to the command block in which they are defined, or any commands blocks nested therein. Variables declared with the var keyword have function scope in that their scope is limited to the function in which they are defined, or any functions nested therein.]

4. What is the difference between a strongly typed and a loosely typed language? Which is JavaScript?
    
	[Variables in a strongly typed language do not change after they have been declared. Programming languages that do not require you to declare the data types of variables are called loosely typed programming languages. A JavaScript is a loosely typed language.]
