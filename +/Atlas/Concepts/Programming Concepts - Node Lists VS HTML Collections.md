Node lists and HTML collections are similar, but there are some important differences. 

## HTML collections
HTML collections can **only contain elements**.

Items within an **HTML collection** can be referenced by: 
- index number
- element id
- or element name. 

HTML collections are **dynamic** so that changes in the structure of the web page will be **automatically reflected** in the HTML collection.

## Node List
Items within a node list **can only be referenced by their index number**. 

 A node list **can contain a variety of node types** including `elements`, `text strings`, and `attributes`. 

A node list is **static**. Once it has been created it **will not automatically update itself** even as the document changes.
