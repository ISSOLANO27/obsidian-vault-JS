list of terms and matching descriptions.
```html
<dl>
    <dt>term1</dt>
    <dd>description1</dd>
    <dt>term2</dt>
    <dd>description2a</dd>
    <dd>description2b</dd>
</dl>

```
dl - description lists
dt - description term
dd - descriptions

> NOTE: descriptions **must always directly follow the term** they describe and that **more than one description may be provided with each term**.

By default most browsers indent the description with each term. Markers are rarely used.
