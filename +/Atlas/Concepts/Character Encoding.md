The process by which the computer **converts text into a sequence of bytes when it stores the text** and **then converts those bytes back into characters when the text is read**.

The most common character encoding is [[UTF-8]], which supports almost all of the characters you will need.

Each character from a character set is associated with an encoding value that can then be stored and read by a computer program.

For example, the copyright symbol `©` from the Unicode character set is encoded with the number 169. If you know the encoding value, you can insert the corresponding character directly into your web page using the following character encoding reference:


```html
&#169;
```
