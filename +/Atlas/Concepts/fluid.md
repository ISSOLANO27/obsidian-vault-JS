A fluid layout sets the width of page elements as a **percent** of the available screen width. 

For example, the width of the page body might be set to fill 90% of the screen and the width of the company logo might be set to fill 80% of that page body. **Under a fluid layout, the page resizes automatically** to match the screen resolution of the user’s device.
