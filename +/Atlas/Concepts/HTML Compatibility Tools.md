

Tools used to support modern HTML features in older browsers.

| Tool | Purpose |
|----|----|
| Modernizr | Detects support for HTML5 and CSS features |
| HTML5 Shiv | Enables HTML5 elements in older versions of Internet Explorer |
| Editor Scripts | Some editors (e.g., Dreamweaver) include built-in compatibility scripts |

Notes:
- These tools help legacy browsers recognize new elements
- Rendering may still differ from modern browsers
