A fixed layout is one in which the size of the page and the size of the page elements are fixed, usually using **pixels** as the unit of measure. 

The page width might be set at 960 pixels and the width of the company logo set to 780 pixels. **These widths are set regardless of the screen resolution of the user’s device** and this can result in the page not fitting into the browser window if the device’s screen is not wide enough.
