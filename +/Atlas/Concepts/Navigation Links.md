an unordered list of hypertext links placed within the [[nav]] element.
```html
<nav>
	<ul>
		<li>Link1</li>
		<li>Link1</li>
	</ul>
</nav>
```

Link 1 and 2 are hypertext links. Although you could place it anywhere...

Having a central list of links makes the website easier to work with and navigate #writingEffectiveCode
