a path that **starts from the root folder** and processes down the entire folder structure described with the expression:
```/folder1/folder2/folder3/filename
```
or 
```/thai/docs/catering/ct_catering.html
```
#absolutePath


---
If [[files are located on different drives]] #whatDoesThisMean as well as in different folders, you **MUST** include the drive letter in the path with the expression: `/drive|/folder1/folder1/folder1/file`
```/drive/folder1/folder2/folder3/file
```

---

Where drive is the letter assigned to the drive. Note that the driver letter **MUST** be followed by the `|` character.

If the driver was located on driver E, the **ABSOLUTE** path that includes the drive would have the E in the expression.
`/E|/thai/docs/catering/ct_catering.html`

>Note that you don’t have to include a drive letter if the linked document is located on the **same drive as the current file**.
