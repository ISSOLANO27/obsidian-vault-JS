Documents on the web are stored on web servers in the form of [[Web Pages]] and accessed through a software program called web browser. 

> The browser retrieves the document from web server and renders it in a form that is readable on a client device.

> web servers translate the results into something called a [[Scripting Engine]] of a script into HTML, which then sends to the client
