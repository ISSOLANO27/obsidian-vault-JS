If a nonnumeric operand cannot be converted to a number, you receive a value of `NaN`. 

The value NaN stands for “**Not a Number**” and is returned when a mathematical operation does not result in a numerical value. 

The sole exception to this rule is `+=`, which **simply concatenates operands if one or both can’t be converted to numbers.** #LooselyTyped #duckTyped
