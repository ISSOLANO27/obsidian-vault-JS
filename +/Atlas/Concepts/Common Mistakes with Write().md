- Naming convention:
	- When talking about the language in writing, built-in objects are referred in capital letter. *ex. Document object, Math object, Array object.* 
	- But when you actually code in JS, the names of those objects are lowercase.
	- ```js
	  document.write() 
	  document.writeln()
	  ```
> make sure to code objects in lowercase.
