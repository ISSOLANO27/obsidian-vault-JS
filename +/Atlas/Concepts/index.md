Indexes always start with the number 0, so the first element in any array has an index of 0, the second has an index of 1, and so forth. 

You can set a specific array value by its index using the expression:

`array[index] = value;`

```js
monthName[0] = "january";
monthName[1] = "February";
monthName[2] = "March";
...
