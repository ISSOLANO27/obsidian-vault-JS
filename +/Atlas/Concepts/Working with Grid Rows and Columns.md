To define the number and size of grid columns use `grid-template-column`
```css

grid-template-columns: width1 width2;`
```

The following creates 2 columns. The first 250px and the other at 100px:
```css
grid-template-columns: 250px 100px;
```
---
### Columns widths can be expressed using
- pixels
- ems
- percentages

	#### Using `auto` to let the Browser decide..
	You can also use `auto` to **allow the column width to be automatically** set by the browser:
```css
grid-template-columns: 100px auto 50px;
```

> This could be used in a layout where the first and third columns are used for navigation lists that are fixed in size while ethe middle column contains the article.

You typically do not define the number of rows and just let the browser automatically add rows as needed to contain all page elements within the grid container

An **explicit** grid completely defines the number and size of the grid rows and columns. 
#explicitGrid
An **implicit** grid contains rows and/or columns that are **generated by the browser** as it populates the grid with items from the grid container. In most grid layouts you will explicitly define the columns and let the browser fill out the grid rows drawn from the web page content. #implicitGrid 

### Explicit Grid
To **explicitly** define the number of rows and their height, use the following:

`grid-template-rows: height1 height2;`

```cs
dic#outer {
	display: grid;
	grid template-columns: 250px 100px;
	grid template-rows: 50px 100px 150px;
}
```

### Implicit Grid
The rows heights are determined by the page element.

You can also set the row heights in an implicit grid using `gris-auto-rows` property:

`grid-auto-rows: height1 height2;`

> TIP: If the content of a grid item is greater than can be displayed **within the allotted height**, the browser will automatically increase the row height to match.

This sets the height of the first row in the implicit grid to 100 pixels, the height of the second row to 200 pixels and then repeats those heights for each subsequent set of 2 rows until the grid is filled.:

```css
grid-auto-rows: 100px 200px;
```
