both **assign a value** and **perform a calculation**. For example, the expression:
```js
x += y
```

| OPERATOR | EXAMPLE | EQUIVALENT TO |
| -------- | ------- | ------------- |
| =        | x = y   | x = y         |
| +=       | x += y  | x = x + y     |
| −=       | x −= y  | x = x − y     |
| *=       | x *= y  | x = x * y     |
| /=       | x /= y  | x = x/y       |
| %=       | x %=y   | x = x % y     |
| **=      | x**=y   | x = x**y      |
|          |         |               |
|          |         |               |
You can use the += compound addition assignment operator to [[combine two strings]] as well as to add numbers.

Before combining operands, a [[JavaScript interpreter attempts to convert a nonnumeric operand, such as a string, to a number]].
