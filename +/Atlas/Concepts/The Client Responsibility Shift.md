Originally the division of labor was very strict.

Job was mostly:
- show an interface
- collect input
- send request
- display results



Devices got powerful. Ridiculously powerful..

"*Why send everything to the server if the client can handle some of it instantly?*"

Browsers could now lift a load from the server. see [[What Moved To The Client]]



---

This is where [[Document Object Model (DOM)]] + JavaScript matter so much
