These elements are those that have behavior not just static. 

For example, embedded audio or video content usually contains player buttons to control the playback.

```html
<img src="file" alt="text" />
```
---
If the Browser cannot render the image, the `alt` attribute is used instead.




>TIP: Always use the alt attribute. It is **REQUIRED** in XHTML code and is highly recommended as a way of accommodating users running nonvisual web browsers #oldCodeNeverDies #writingEffectiveCode
