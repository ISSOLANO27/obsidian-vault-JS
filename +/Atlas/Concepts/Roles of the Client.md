One of main roles of the **client (front end)** is the `presentation to the user`. 

The user interface gather the information from the user, submits it to the Web Servers (back end)**, then receives, formats, and presents the results return from the server.

The Web Browser is responsible for formatting and presenting the document to the user via[[Hypertext Transfer Protocol (HTTP)]].
