reasons for this:

- The code can be **shared among multiple pages**.
    
- Because the code is shared among multiple pages, it only **needs to be downloaded once**, which is a great benefit for users on mobile connections.
    
- Team members can more **easily share code** for joint projects.
    
- The HTML file will be **neater and cleaner** because your HTML elements will not be combined with JavaScript codes.
    
- It is easier to manage your website if each file focuses on a single task: 
	- HTML files for document content and structure
	- CSS files for page design and layout
	- JavaScript files for interactive features and specialized tasks. 
	
	 **Team members would then be responsible only for those files** within their area of expertise.
