## How much processing should I place on the client and how much should I place on the server?

Sometimes the answer is simple..

- Control over the browser = JavaScript
- Access to a database = MUST use a server-side scripting language.

Additionally, [[There Are Tasks That BOTH Can Perform]]. A general Rule of Thumb, is to [[Split the Labor]] 

Specially when dealing with clients and servers over the web. 

> If you want to access a database on the `web browser`, you **MUST** use a server-side script..
