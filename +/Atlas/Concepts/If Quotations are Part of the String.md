simply use double quotation marks to enclose single quotes and single quotation marks to enclose double quotes. 

The following statements demonstrate this technique by showing how to include first a single quote (') and then a double quote (") as part of a text string:
```js
document.write("Welcome to 'Fan Trick photography'");
document.write('Welcome to "Fan Trick photography"');
// Welcome to 'Fan Trick photography'
// Welcome to "Fan Trick photography"
```
