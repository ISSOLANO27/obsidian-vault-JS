before you can use a variable you need to **declare** it, this creates the variable for storing data and objects.

There are 3 options to **initialize** a variable:
1. using [[let keyword]]
2. Using [[var keyword]]
3. using **[[const keyword]]**

```js
let variable = value;
var variable = value;

const variable = value;
```

> Where **_variable_** is the **name** of the variable
>  and  
> **value** is the **initial value** of the variable.

**=** is known as the **assignment operator**

The value you assign to a variable can be a [[Text String or Literal String]] enclosed within quotation marks or a **numeric value**.

```js
let texRate = 0.05;
let taxClass = "sales";
```
