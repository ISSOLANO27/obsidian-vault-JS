
A Web Framework is just a structured toolkit. It is NOT a language. It is a collection of libraries (pre-written code), conventions (rules about where things go), and tools that help you build web application without reinventing the wheel every time.

Now the key split:

Front-end framework = runs in the browser.  
Server-side (back-end) framework = runs on a server.

Same word “framework.” Different battlefield.
