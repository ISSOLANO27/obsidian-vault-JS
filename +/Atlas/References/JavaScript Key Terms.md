## Introduction to JavaScript

| Term                                    | Meaning                                                                                               |
| --------------------------------------- | ----------------------------------------------------------------------------------------------------- |
| JavaScript                              | A programming language used to add interactivity to web pages                                         |
| Client-Side                             | Code that runs in the user’s browser                                                                  |
| programming language                    | a set of instructions directing the actions of a computer device.                                     |
| compiled                                | process by which those instructions are transformed into machine code.                                |
| markup language                         | defines the content, structure, and appearance of a document. HTML/CSS                                |
| DOM                                     | describes how to access the contents of the web page and user actions within that page.               |
| BOM                                     | describes how to access the features and behaviors of the browser itself.                             |
| Application PRogramming Interface (API) | a set of procedures that access an application such as a web page or a web browser                    |
| Hypertext Transfer Protocol (HTTP)      | main system used on the web for exchanging data.                                                      |
| Validation                              | Checking user input for correctness                                                                   |
| Scripting Language                      | A language designed to automate tasks within another environment.                                     |
| static                                  | web pages that do not change once browser renders it.                                                 |
| web application                         | a program that is executed on a server but is accessed through a web page loaded in a client browser. |
| Object                                  | a programming code and data that can be treated as an individual unit or component                    |
| procedure                               | Individual statements used in a computer program are often grouped into logical units.                |
| method                                  | The procedures associated with an object                                                              |
| property                                | a piece of data, such as a color or a name, which is associated with an object.                       |
| variables                               | the value a program stores in computer memory                                                         |
