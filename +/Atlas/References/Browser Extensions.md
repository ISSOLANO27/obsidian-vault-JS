In addition to the W3C-supported style properties, most browsers supply their own extended library of [[browser style sheet]] properties, known as **browser extensions**. 

Many of the styles that become part of the W3C specifications start as browser extensions and for older browser versions, sometimes the only way to support a particular CSS feature is through a browser extension tailored to a particular browser.

Browser extensions are identified through the use of a [[vendor prefix]]. indicating the browser vendor that created and supports the property.
