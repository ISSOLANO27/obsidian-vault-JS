

| Year | Event | Why It Matters |
|----|----|----|
| 1989 | HTML 1.0 introduced | First public version of HTML; basic document structure |
| 1995 | HTML 2.0 | Added forms and basic interaction |
| 1997 | HTML 3.2 | Introduced tables and scripting support |
| 1999 | HTML 4.01 | Added CSS support and multimedia elements |
| 2001 | XHTML 1.0 | HTML rewritten using XML rules for strictness |
| 2002 | XHTML 1.1 | Minor refinement of XHTML |
| 2004 | WHATWG formed | Reaction against XHTML 2.0’s strict direction |
| 2009 | XHTML 2.0 abandoned | Failed due to lack of backward compatibility |
| 2012 | HTML 5.0 | Modern HTML with backward compatibility |
| 2017 | HTML 5.2 | Current recommended version |
