JavaScript treats all numeric values as binary values, rather than as decimals—that is, the **numbers are calculated using** the [[two-digit binary system]] rather than the [[10-digit decimal system]]. 
While the binary system can accurately represent any value that has a decimal equivalent, when it comes to floating point values, [[Calculations performed on Binary representations can result in slightly different results than the same calculations performed on decimal values]].
