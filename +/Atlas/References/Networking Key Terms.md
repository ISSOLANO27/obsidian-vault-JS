# Networking Key Terms

| Term | Meaning |
|----|----|
| Network | A group of devices that share information and resources. |
| Node | Any device connected to a network. |
| Host | A device capable of sending or receiving data. |
| Client | A device that requests data or services. |
| Server | A device that provides data or services to clients. |
| LAN | A local area network, usually confined to one building or location. |
| WAN | A wide area network that connects multiple LANs across distances. |
| Internet | A global system of interconnected networks. |
| Protocol | A set of rules that define how data is exchanged. |
| Packet | A small unit of data sent over a network. |
