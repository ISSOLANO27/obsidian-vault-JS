Element used to embed code in order to add JavaScript to a web page.

```js
<script>
	statements
</script>
```
where *statements* are the individual lines of code in the JavaScript Program

The following **script** contains a single statement displaying an alert window with the text message "*Hello World*".

```js
<script>
	window.alert("Hello World");
</script>
```
