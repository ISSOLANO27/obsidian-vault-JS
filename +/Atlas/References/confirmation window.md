A window that does provide the ability for the user to respond is a [[confirmation window]] that displays a message along with an OK and Cancel button. The window returns a value of true or false depending on whether the user closed the window by clicking the OK button or the Cancel button. To create a confirmation window, apply the following `window.confirm()` method:
```js
response = window.confirm(message)
```

### `window.confirm()` Stored as Boolean
where **_response_** is a Boolean value that is **true** if the user clicks the OK button 
and 
**false** if the user clicks the Cancel button, and _message_ is the text that appears in the window.
