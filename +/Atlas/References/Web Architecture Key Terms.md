# Web Architecture Key Terms

| Term | Meaning |
|----|----|
| World Wide Web | A system of interlinked documents accessed through the Internet. |
| Web Page | A single document on the web, usually written in HTML. |
| Website | A collection of related web pages. |
| Web Browser | Software that retrieves and displays web pages. |
| Web Server | A computer that stores and delivers web pages to clients. |
| URL | The address used to locate a resource on the web. |
| HTTP | The protocol used to transfer web pages and resources. |
| Hyperlink | A clickable reference that navigates to another resource. |
| Client–Server Model | A system where clients request resources from servers. |
| Rendering | The process of turning code into a visible page. |
