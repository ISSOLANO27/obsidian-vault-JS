Besides a scripting language, JS is also considered an object-based programming language.

An `object` is programming code and data can be treated as an individual unit or component. 

Individual statement used in a computer are often grouped into logical unites called [[procedures]].

`carLoan` object could do 2 things:
1. calculate a number of payments ([[method]])
2. store information ([[property]])
