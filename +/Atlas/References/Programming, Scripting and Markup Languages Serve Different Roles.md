

Web development uses different types of languages, each with a specific job.

[[Programming Languages]] are used to build software itself.
They must be compiled into machine code before they can run.

[[Scripting Languages]] are executed at runtime by an interpreter. No need for a compiler.
They allow programs to respond dynamically without a separate build step. 

Scans for error even as it runs.. Every browser has a JavaScript interpreter.



[[Markup Language]] describe structure and appearance rather than behavior.
Modern websites rely on all three working together rather than a single language doing everything.
