A variable stores values in computer memory.

Data stored in a specific variable often will change.

How they work:
1. Write a statement that creates the variable and assigns it a name. ex. `curTime` (*stores the current time in that variable*).

> Each time that the program runs, the current time is a different time. Value varies.

Programmers refer to storing a value in a variable as *"Assigning a value to a variable"*. #programmingJargon
