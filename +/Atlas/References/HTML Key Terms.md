
# HTML Key Terms

| Term                          | Meaning                                                                                                                                                                    |
| ----------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| HTML                          | The markup language used to structure content on the web.                                                                                                                  |
| Element                       | A building block of HTML, usually made of a tag and content.                                                                                                               |
| Tag                           | A marker used to define an element (e.g., `<p>`, `<a>`).                                                                                                                   |
| Attribute                     | Extra information added to an HTML element (e.g., `href`, `class`).                                                                                                        |
| Hypertext                     | Text that links to other documents or resources.                                                                                                                           |
| Markup Language               | A language that describes structure and meaning, not behavior.                                                                                                             |
| Deprecated                    | A feature that is no longer recommended but still supported.                                                                                                               |
| Backward Compatibility        | The ability for newer systems to support older content.                                                                                                                    |
| HTML5                         | The modern version of HTML designed to evolve without breaking the web.                                                                                                    |
| XHTML                         | A stricter, XML-based attempt to reform HTML that was ultimately abandoned.                                                                                                |
| Quirks mode                   | Absence of Declaration practice (1900s-2000s)                                                                                                                              |
| Standard mode                 | Employed the latest specifications                                                                                                                                         |
| Meta data                     | Content that describes the document or provides information about how the document should be processed by the browser                                                      |
| Character Encoding            | The process by which the computer converts text into a sequence of bytes when it stores the text and then converts those bytes back into characters when the text is read. |
| Semantics/Sectioning Elements | Elements whose tags describes the purpose of the element and the type of content it contains.                                                                              |
| Embedded content              | content imported from another resource, often nontextual, such as graphic images, audio soundtracks, video clips, or interactive games                                     |
| Polyfills                     | teach older browsers how to handle newer features                                                                                                                          |
| scripts                       | elements without an external program that runs in the browser                                                                                                              |
