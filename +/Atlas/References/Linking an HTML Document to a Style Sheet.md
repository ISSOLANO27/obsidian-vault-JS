A **stylesheet** is a set of rules that specify how page elements are displayed using a language calls Cascading Style Sheets (CSS). 

Was developed and enhanced as the web grew and changed.

Managed by W3C.

To link use:

```html
<link href="file" rel="stylesheet">
```

`file` is the text file containing the style sheet.

Since `link` can be used to link to other data, the `rel` attribute is required to tell the browser what it is linking to. 

>Older websites might contain `type="text/css"`

Link is metadata, it ALWAYS belongs in the head.
