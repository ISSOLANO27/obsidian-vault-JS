Concept: The Dom is an object-based representation of the document that allows code to:
- Access structure
- Respond to user events
- Modify content and style

The `document` object provides methods that allow JavaScript to **search**, **modify**, and **create elements** within that structure. 

These methods return element objects or collections of elements. #HTMLCollections
