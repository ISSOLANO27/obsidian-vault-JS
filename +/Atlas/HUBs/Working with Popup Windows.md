
Many applications require an external window to be opened and displayed on top of or adjacent to the application content. Within these windows, also known as [[popup windows]], the application can include additional information to users or provide an area where users can provide feedback that can be used by the application. 

For example, a popup window might prompt a user for a password or email address.
