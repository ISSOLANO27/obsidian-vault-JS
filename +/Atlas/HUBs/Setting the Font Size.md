Another important consideration in [[Typography]] is the text size, which is defined using the following font-size property:

`font-size: size;`

where _size_ is a length in a CSS unit of measurement. 

Size values for any of these measurements can be whole `numbers` ` (0, 1, 2 …)` or `decimals (0.5, 1.6, 3.9 …)`. 

Lengths (and widths) in CSS are expressed in either absolute units or relative units.
