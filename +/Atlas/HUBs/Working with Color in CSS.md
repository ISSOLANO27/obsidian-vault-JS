The first part of your style sheet for the Tri and Succeed Sports website will focus on color. If you’ve worked with graphics software, you’ve probably made your color selections using a graphical interface where you can see your color options. 

Specifying color with CSS is somewhat **less intuitive** because CSS is a text-based language and requires colors to be defined in textual terms. This is done through a **color name** or a **color value**.
