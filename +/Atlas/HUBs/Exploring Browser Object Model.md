
In some situations, an app may need to work with the user’s browser or device. In this section you will review some of the objects, properties, and methods associated with the Browser Object Model (BOM). 

Like objects in the Document Object Model, objects in the BOM are also arranged in a hierarchy with the browser window at the top of the tree.

![[Browser Object Model..png]]

The browser **automatically creates** the window object and the other objects in tree. 

The window object is also referred to as a **global object** because all other objects in the BOM are contained within it.
