```css
	h1 {
		color: red;
	}
```
This will match all h1 elements regardless of the location within the page.

You will often want to style rules to apply to elements that are only placed within specific location. like heading within an article and NOT ANYWHERE ELSE. 

in this case use, **selector patterns** to match only those page elements that match a specific pattern.
