
An [[array]] is a set of data represented by a single variable name. You use an array when you want to store a group or a list of related information in a single, easily managed location. Lists of names, courses, test scores, and prices are typically stored in arrays.


> The identifier you use for an array name must follow the **same rules as identifiers for variables.** It must begin with an uppercase or lowercase ASCII letter, dollar sign ($), or underscore (_), can include numbers (but not as the first character), cannot include spaces, and cannot be reserved words.
