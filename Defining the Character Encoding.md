As with HTML files, it is a good idea in every CSS document to define the character encoding used in the file. In CSS, you accomplish this using the following` @charset` rule:
```css
@char "encoding";
```

where `encoding` **defines the character encoding used** in the file. 

Add the @charset rule to the tss_styles.css style sheet file now, specifying that the UTF-8 character set is used in the CSS code.
